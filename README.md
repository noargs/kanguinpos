### docs
- download and import to your idea as existing project
- some functionalities are not developed yet i.e. Back Office(Admin), etc
- cardlayout to switch between `com/swingpos/ui` on single JFrame
- `com/swingpos/main/PosWindow.java` to control all the `com/swingpos/ui`
- miglayout as a main layout
- `com/swingpos/controller` to fetch from db through `com/swingpos/dao`
- `com/swingpos/model` to create and manipulate LinkedList
- Category and Product one to many relationship
- Order and Item one to many relationship (i.e. for JTable)
